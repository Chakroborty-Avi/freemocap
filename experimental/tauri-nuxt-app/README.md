# Tauri Nuxt App

## Installation
- Install Rust on your system - https://www.rust-lang.org/tools/install 
- Install Node.js
- Run the following commands in the root directory (i.e. the directory containing `package.json`, `src-tauri/` and `src-nuxt/`, etc.) run:

```
npm install
```

## Running the app
```
npm run tauri-dev
```

whee!
